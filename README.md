# 🎄 Christmas Tree

一个简单的圣诞树HTML页面，包含自动播放的圣诞视频。

## 📁 项目结构

```
.
├── README.md           # 项目说明文档
├── christmas_tree.html # 主HTML文件
├── content.mp4         # 圣诞视频文件
└── .gitignore          # Git忽略配置
```

## 🚀 使用方法

1. 克隆仓库：
   ```bash
   git clone https://github.com/yinyuscloor/ChristmasTree.git
   ```

2. 直接在浏览器中打开 `christmas_tree.html` 文件

3. 页面将自动播放圣诞视频

## 🎯 功能特点

- **自动播放视频**：页面加载后自动播放圣诞视频
- **全屏显示**：视频自适应浏览器窗口大小
- **循环播放**：视频循环播放，营造节日氛围
- **简洁设计**：无多余元素，专注于节日体验

## 🛠️ 技术栈

- **HTML5**：页面结构
- **CSS3**：样式布局
- **JavaScript**：视频控制
- **MP4视频**：节日内容

## 📝 文件说明

### christmas_tree.html
主HTML文件，包含：
- 全屏视频播放器
- 自动播放和循环设置
- 响应式布局

### content.mp4
圣诞主题视频文件，约6.5MB，包含节日氛围内容。

## 🔧 自定义

如果你想自定义这个项目：

1. **更换视频**：替换 `content.mp4` 文件为你自己的视频
2. **修改标题**：在HTML文件中修改 `<title>` 标签
3. **添加样式**：在 `<style>` 标签中添加自定义CSS
